My Projects

Throughout my journey of developing valuable and relevant technical skills, I have created many projects with a meaningful real-world application

  • Supervise

    Supervise is inventory manager project was recently developed using a combination of React, JavaScript, Node.js, Express, and MySQL. The project aimed to streamline the process of managing inventory for a retail business. The user interface was built using React, which provided a responsive and intuitive design for users to interact with. JavaScript was used for the core logic and functionality of the application, while Node.js and Express were used to handle server-side operations such as data storage and retrieval. MySQL was used as the database management system to store and organize the inventory information.

  • ShareTrace

    ShareTrace was developed using Python, BeautifulSoup, Requests, and Microsoft Excel. The project aimed to track inventory stock information for a business and store it in an organized manner. Python was used for the core logic of the program and to automate the process of retrieving stock information from the web. BeautifulSoup and Requests were used to scrape and extract the required information from various websites. The extracted information was then stored in a Microsoft Excel spreadsheet, which allowed the business to easily view and analyze the inventory stock data.
